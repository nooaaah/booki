# Booki

Ce travail a été réalisé dans le cadre du projet 2 de la formation Développeur Web d’OpenClassrooms.

Ce repository contient le code du prototype du site Booki.  
Booki est une entreprise fictive permettant à ses utilisateurs de planifier leurs vacances en trouvant des hébergements et des activités dans la ville de leur choix.

Les utilisateurs peuvent filtrer les résultats selon différentes thématiques afin d’affiner leur recherche.  
L’objectif principal de ce projet était d’intégrer une interface responsive à partir d’une maquette fournie.

## Technologies utilisées
- HTML5
- CSS3
- Responsive Design
